Technology II

By Valentin Dirken

Course content

In this course it was about gaining an understanding of the principles and applications of serverless functions through 8 sessions of 3 hours each. Using tools such as Lambda and other services within Amazon Web Services, we learned how to design serverless applications that can retrieve data from sensors, process it, and present it in a meaningful way. Moreover, this course aimed to provide us with a foundational knowledge of cloud computing and the ability to apply this knowledge to real-world scenarios.

My expierence

It was a very practical course that learned us to understand html, buckets, API calls, Lambdas and databases. It was very hands-on, a lot of try and error. For me this way of learning worked very good. But the tempo was rather high and experimenting at home was a must to keep up. We had to make a dynamic webpage with a database where you could add some data. I was able to work in a very motivated group and we made a very cool site with an add, delete and update function.


Grading
